# Birthday Wish (Mobile)

## Run
- **Option A (recommended)**: start a local server from this folder:

```bash
python -m http.server 5173
```

Then open `http://localhost:5173` on your phone (same Wi‑Fi) or on your PC.

- **Option B**: open `index.html` directly (works in most browsers, but a server is smoother).

## Usage
- **PIN**: `2407`
- After unlocking, the **Home** page shows the background + confetti burst.
- Tap **Gift** → Gift page 1 (zoom), then **click to see buety.** → Gift page 2 (slide with stars).
- Tap **Letter** → Letter page (zoom with stars).
- On Gift/Letter pages, tap the **“wonea go home”** circle to return.
